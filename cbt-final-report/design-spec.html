Mobile Strategy

  • 📱 Filter chips → "필터 (3)" trigger → bottom sheet
  • 📱 Tabs → horizontal scrollable tab strip, active tab centered
  • 📱 Similar companies → swipe carousel (1 card visible)
  • 📱 Tables → 2-3 key columns + horizontal scroll, or card view
  • 📱 Network graphs → default to list view + "그래프로 보기" toggle
  • 📱 Density → default Comfortable (desktop: Compact)
  • 📱 Breakpoint: 768px threshold (CDS)

KSIC 업종 매칭 + 로그 스케일 매출 유사도 + 지역 매칭. Phase 1 Simple Matching.

Design Principles

🎯 Satisficer vs. Maximizer

Satisficers: Top 1-2 "Best Match" 강조 카드 → 첫 번째 좋은 옵션 클릭.

Maximizers: 전체 비교 테이블, PitchBook-style side-by-side → 모든 옵션 평가 후 결정.

📊 Intelligent Density

High information per screen, but with clear visual hierarchy. Typography weight, spacing, strategic bolding. Korean text: weight 400, line-height 1.6-1.8x.

🔐 Progressive Disclosure = Paywall

Layer 0 무료 → Layer 1 기본 96🍪 → Layer 2 심화 456🍪. "Show shape, hide numbers" — structure visible, values blurred.

🚫 Dead-End 제거

모든 페이지에 다음 탐색 경로 제공. 결과 0건 시 "이 업종의 인기 기업" 등 대안 콘텐츠.

💬 추천 근거 투명성

"왜 이 기업이 추천됐는지" 사용자가 이해 가능해야 신뢰. Match reasoning chips 필수.

🤖 AI suggests, User confirms

Phase 2에서 AI 필터 변환 시 — 자동 적용 금지. 파싱된 chip 보여주고 "이 조건으로 검색" 확인 버튼.

Phase 2 · AI-Native Vision

Phase 2-A · Conversational Search → Structured Filters

Natural language → structured filter conversion. "부산 핀테크 스타트업 직원 50명 이상 매출 100억 미만" → editable chips.

5 States: Parsing → Filter confirmation → Disambiguation → Conversational refinement → Error recovery

Phase 2-B · AI-Generated Summaries & Smart Recommendations

Perplexity-style cited summaries (Claude Sonnet 4.5). AI company fit scoring from search log data. Three recommendation types:

  • "이 기업을 본 사용자들이 함께 본 기업" — collaborative filtering
  • "관심 업종 트렌드" — weekly digest
  • "네트워크 인텔리전스" — relationship signals (Harmonic.ai pattern)
Phase 2 · Thumbs Up/Down Feedback Loop

👍/👎 on similar company cards → stored in MongoDB → feeds Phase 2 Hybrid Scoring. Personal recommendation weight adjustment per user.
